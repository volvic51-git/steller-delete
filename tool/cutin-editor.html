カットイン台本エディター（data/cutin/*.json）

プロジェクト:
立ち絵ファイルは assets/images/cutin/ に置く想定です（幅600px前後・バストアップ）。 命名規則の目安: cutin_<charId>_<expression>.png。 このエディタは同フォルダへの相対パスで画像を読み込むため、未配置のうちはサムネイルが「?」表示のままで構いません。
トリガー種別のうち time / open_rate / mines_removed / manual はプレイ中に発火し得るため、そのステージはランキング対象外になります （stage_start / stage_clear のみなら対象のまま）。
書き出し設定
data/cutin/<セット名>.json としてダウンロードされます
assets/audio/ から読み込みます。行ごとに個別指定・無音固定で上書き可能
