Stellar Delete — Board Hunter
=================================

「保証盤面（推測なしで必ず解ける盤面）」をみんなで探すツールです。

■ あそびかた
  1. index.html をブラウザで開く（インターネット接続は不要）。
  2. 探索ターゲットを選んで「探索スタート」を押す。
  3. あとは放置。見つかった盤面の seed が自動でたまります。
  4. ときどき「結果を保存」を押して、できたファイルを開発者に送ってください。

■ 送るのは「seed の数字」だけ
  盤面そのものは送りません。開発者が同じ seed から盤面を再生成し、
  あらためて検証してから公式に採用します。だからデータはとても軽く、
  改竄しても再検証で分かる仕組みです。

■ ファイル構成（配布物）
  board-hunter/
  ├── index.html   … 本体（これを開く）
  ├── solver.js    … ソルバー（ゲーム本体と共通のロジック）
  └── README.txt   … このファイル

  ※ 3つを同じフォルダに置いたまま使ってください。

■ 結果の送り方
  「📧 メールで送る」ボタンを押すと、メーラーが自動で開きます。
  そのまま送信してください。（宛先: stellard26@gmail.com）

  seed が 100件を超えた場合は「JSONを保存」でファイルを保存して、
  メールに添付してください。

■ 注意
  - タブを開いている間だけ探索が進みます。閉じると止まります。
  - 大きい盤面ほど 1 回の判定に時間がかかり、見つかりにくくなります。
  - 同じ seed を別の人が見つけても問題ありません（開発者側で重複を除きます）。
